# GitHub Followers

This project is based off Sean Allen's course. It creates an iOS app that uses the GitHub API to retrieve followers for a given account plus details for each of the accounts.

## How to install

1. fork the project or download the .zip
2. open project in XCode v15 or later (earlier versions may or may not compile correctly)
3. run in simulator
